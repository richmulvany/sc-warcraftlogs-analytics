Gold layer table definitions. These are the tables exported to the frontend.

## gold_entity_summary

**Description:** Aggregated summary of entities by category. One row per category.

**Updated:** Nightly via DLT pipeline + export job.

**Source:** `silver_entities`

| Column | Type | Description |
|--------|------|-------------|
| `category` | STRING | Entity category name |
| `total_count` | LONG | Total number of entities in this category |
| `unique_count` | LONG | Count of distinct entity IDs |
| `latest_created_at` | TIMESTAMP | Most recent entity creation timestamp |
| `earliest_created_at` | TIMESTAMP | Oldest entity creation timestamp |
| `_gold_generated_at` | TIMESTAMP | When this row was generated by the pipeline |

**Known limitations:**
- Soft-deleted entities are not excluded (no `deleted_at` field in source)
